Dating App Server

Available endpoints:

  • GET / - This page
  • GET /eventbrite/authorize - Start Eventbrite OAuth flow
  • GET /eventbrite/config - Show exact redirect URI, hostname, and authorize URL being used
  • GET /eventbrite/callback - OAuth redirect endpoint (expects ?code=...)
  • GET /eventbrite/categories - Fetch all Eventbrite categories
  • GET /eventbrite/events - Fetch events by category and date range (supports page + pageSize)
